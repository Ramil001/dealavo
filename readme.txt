== Description ==

Dynamic Price with Dealavo is a WordPress plugin that connects your WordPress e-commerce website with Dealavo service to dynamically update sales prices based on your Dealavo pricing recommendations. This plugin provides an automated way to keep your product prices competitive and optimized.

== Installation ==

1. Upload the `dynamic-price-with-dealavo` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Dealavo Settings' in the WordPress admin dashboard to configure your Dealavo API key, Account ID, and update time.

== Frequently Asked Questions ==

= How do I obtain a Dealavo API key and Account ID? =

You can obtain your Dealavo API key and Account ID by signing up for an account on the Dealavo platform. Once you've signed up and created an account, you'll find your API key and Account ID in your account settings.

= How often are prices updated? =

By default, prices are updated daily. You can customize the update time in the plugin settings.
